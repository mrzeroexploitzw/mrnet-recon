# MRNET RECON PRO v5.0

MRNET RECON PRO is a multi-threaded network reconnaissance toolkit built for fast and efficient information gathering.

---

## 🔥 Features

- Subdomain Scanner (crt.sh based)
- Alive Host Detection
- Fast Multi-threaded Port Scanner
- DNS Lookup
- SSL Certificate Inspection
- HTTP Header Intelligence
- IP Information Lookup
- Reverse DNS Lookup
- Banner Grabbing
- Zerorated Website Finder (by country)

---

## ⚙️ Requirements

- Python 3.x
- Internet connection

### Python Modules:
- requests
